课题介绍：
近年来大模型技术（如 GPT、LLM、VLM、AI Agent）爆发，展现出了人工智能前所未有的归纳总结能力和泛化能力。基于 Agent 的 AI 搜索正在各行各领域迅速验证其对越来越多相对纯人力劳动，及至于传统搜索引擎更高效的问题解决能力，也是当下行业研究最热门的领域之一。
字节跳动有着超大规模的用户群，搜索团队一直在用最前沿的技术服务着线上海量的互联网用户消费意图。同时，我们也在紧跟时代的脚步，持续探索 AI 搜索如何在已有的业务场景，以及潜在的新领域更好的服务于广大中国互联网网民。
本课题旨在探索业界最新的基于大模型的 AI Agent 技术，尤其是如何落地应用于现有的服务数亿用户的互联网场景，产生相比传统搜索引擎显著更优的增量用户价值，使广大互联网用户的需求得到更好的满足。
1、对 AI Agent / AI 搜索架构进行创新性研究，探索业界架构全新范式；
2、推进大模型与应用结合，将 Al Agent 在大规模互联网场景下落地；
3、不断提升架构灵活性，提升 Agent 流程编排能力；
4、提升模型 ReAct 式框架调度能力上限；
5、通过全链路和局部 (CPU + GPU) 的性能优化，不断提升 Agent 的执行性能。
